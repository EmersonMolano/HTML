Etiquetas de Listas

<ul>

Lista desordenada

Definición: Crea una lista de ítems sin orden específico. Cada ítem se muestra con una viñeta (•) por defecto. Úsala cuando el orden no importa.

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
  • HTML
  • CSS
  • JavaScript

💡 Ideal para: ingredientes, características, opciones, menús de navegación.

<ol>

Lista ordenada

Definición: Crea una lista numerada donde el orden importa. Puede personalizar el número inicial con start y el tipo de numeración con type.

<!-- Lista numérica -->
<ol>
  <li>Instalar el editor</li>
  <li>Crear index.html</li>
  <li>Abrir en el navegador</li>
</ol>

<!-- Lista con letras, desde C -->
<ol type="A" start="3">
  <li>Opción C</li>
  <li>Opción D</li>
</ol>

Pasos:

  1. Instalar el editor
  2. Crear index.html
  3. Abrir en el navegador

Con letras:

  1. Opción C
  2. Opción D
<li>

Ítem de lista

Definición: Define cada elemento dentro de una lista <ul> u <ol>. Puede contener texto, imágenes, enlaces u otras listas anidadas.

<ul>
  <li>Elemento simple</li>
  <li><a href="#">Elemento con enlace</a></li>
  <li>Elemento con lista anidada
    <ul>
      <li>Sub-elemento 1</li>
      <li>Sub-elemento 2</li>
    </ul>
  </li>
</ul>
  • Elemento simple
  • Elemento con enlace
  • Elemento con lista anidada
    • Sub-elemento 1
    • Sub-elemento 2
