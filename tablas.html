Etiquetas de Tablas

<table>

Tabla de datos

Definición: Contenedor principal que define una tabla de datos. Dentro van <thead> para los encabezados y <tbody> para los datos. Solo debe usarse para información tabular, nunca para maquetar páginas.

<table>
  <thead>
    <tr>
      <th>Nombre</th>
      <th>Tipo</th>
      <th>Año</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>HTML5</td>
      <td>Marcado</td>
      <td>2014</td>
    </tr>
  </tbody>
</table>
LenguajeTipoAño
HTML5Marcado2014
CSS3Estilos2011
JavaScriptProgramación1995

Nunca uses <table> para diseñar columnas o layouts. Para eso usa CSS Grid o Flexbox.

<tr>

Fila de tabla

Definición: Define una fila horizontal dentro de la tabla. Contiene celdas de encabezado <th> o celdas de datos <td>. Las celdas pueden fusionarse con colspan y rowspan.

<tr>
  <th scope="col">Producto</th>
  <th scope="col">Precio</th>
</tr>
<tr>
  <td colspan="2">Agotado temporalmente</td>
</tr>
ProductoQ1Q2
TecladoAgotado (colspan=2)
Mouse$35.000$30.000

Usa scope="col" en <th> para indicar que el encabezado corresponde a esa columna.
