Etiquetas Estructurales

<html>

Elemento raíz del documento

Definición: Es la etiqueta raíz que envuelve todo el contenido del documento HTML.

<!DOCTYPE html>
<html lang="es">
  <head>...</head>
  <body>...</body>
</html>

Todo el contenido va dentro de <html>. El atributo lang="es" indica el idioma del documento.

<header> <nav> <main> <section> <article> <aside> <footer>

Elementos semánticos de sección

Definición: Dividen la página en regiones con significado: encabezado, navegación, contenido principal, secciones, artículos, contenido lateral y pie de página.

<header>
  <h1>Mi Sitio</h1>
  <nav>
    <a href="/">Inicio</a>
    <a href="/blog">Blog</a>
  </nav>
</header>
<main>
  <section>
    <article>
      <h2>Artículo</h2>
      <p>Contenido...</p>
    </article>
  </section>
  <aside>Contenido relacionado</aside>
</main>
<footer>© 2025</footer>
<header> — Encabezado
<main> → <section> → <article>
<aside>
<footer>

Etiquetas de Texto

<h1> — <h6>

Encabezados

Definición: Definen la jerarquía de títulos, del más importante (h1) al menos relevante (h6). Son clave para SEO y accesibilidad.

<h1>Título principal</h1>
<h2>Subtítulo de sección</h2>
<h3>Apartado</h3>
<h4>Sub-apartado</h4>

H1 — Título Principal

H2 — Subtítulo

H3 — Apartado

H4 — Sub-apartado

<p> <strong> <em> <mark> <code> <a> <blockquote>

Párrafo, énfasis y citas

Definición: Etiquetas de nivel de frase para añadir significado: importancia, énfasis, resaltado, código en línea, enlaces y citas.

<p>Aprende <strong>HTML5</strong> con énfasis
<em>semántico</em>. Usa <mark>resaltado</mark>.
El atributo <code>href</code> crea enlaces.</p>

<a href="https://ejemplo.com" target="_blank"
   rel="noopener noreferrer">Visitar sitio</a>

<blockquote cite="https://fuente.com">
  <p>"El conocimiento es poder."</p>
</blockquote>

Aprende HTML5 con énfasis semántico. Usa resaltado. El atributo href crea enlaces.

→ Visitar sitio externo

Etiquetas de Listas

<ul> <ol> <li>

Listas ordenadas y desordenadas

Definición: <ul> crea listas con viñetas; <ol> listas numeradas; <li> define cada ítem.

<!-- Desordenada -->
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<!-- Ordenada -->
<ol>
  <li>Instalar editor</li>
  <li>Crear index.html</li>
  <li>Abrir en el navegador</li>
</ol>

Desordenada:

  • HTML
  • CSS
  • JavaScript

Ordenada:

  1. Instalar editor
  2. Crear archivo
  3. Ver en navegador
<dl> <dt> <dd>

Lista de definición

Definición: Agrupa pares término–descripción. Ideal para glosarios y especificaciones técnicas.

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>

  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>

Etiquetas de Formularios

<form> <input> <label>

Formulario básico

Definición: <form> agrupa controles de entrada. <input> captura datos según su atributo type. <label> asocia texto descriptivo al control.

<form action="/enviar" method="POST">
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre"
         name="nombre" required
         placeholder="Tu nombre">

  <label for="email">Correo:</label>
  <input type="email" id="email" name="email">

  <input type="submit" value="Enviar">
</form>
<textarea> <select> <option> <button>

Controles avanzados

Definición: <textarea> para texto largo; <select> para menús desplegables; <button> para acciones más flexibles.

<textarea name="mensaje" rows="4"></textarea>

<select name="nivel">
  <option value="basico">Básico</option>
  <option value="avanzado">Avanzado</option>
</select>

<button type="submit">Enviar</button>

Etiquetas de Tablas

<table> <thead> <tbody> <tr> <th> <td> <caption>

Tabla de datos completa

Definición: Estructuran datos tabulares. <thead> para encabezados, <tbody> para datos, <th> para celdas de cabecera y <td> para datos.

<table>
  <caption>Tecnologías Web</caption>
  <thead>
    <tr>
      <th scope="col">Lenguaje</th>
      <th scope="col">Tipo</th>
      <th scope="col">Año</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>HTML5</td>
      <td>Marcado</td>
      <td>2014</td>
    </tr>
  </tbody>
</table>

Etiquetas Multimedia

<img> <figure> <figcaption>

Imagen con contexto

Definición: <img> inserta imágenes. <figure> y <figcaption> dan contexto semántico relacionando la imagen con su descripción.

<figure>
  <img src="foto.jpg"
       alt="Descripción de la imagen"
       width="800" height="450"
       loading="lazy">
  <figcaption>
    Fig. 1 — Diagrama de HTML
  </figcaption>
</figure>
<video> <audio> <source>

Video y audio nativos

Definición: Permiten embeber medios sin plugins. <source> ofrece múltiples formatos para compatibilidad entre navegadores.

<video controls width="640" poster="mini.jpg">
  <source src="video.mp4"  type="video/mp4">
  <source src="video.webm" type="video/webm">
  Tu navegador no soporta HTML5 video.
</video>

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>
<details> <summary>

Contenido desplegable

Definición: Crea un acordeón interactivo sin JavaScript. <summary> es el encabezado visible; el resto se muestra u oculta al hacer clic.

<details>
  <summary>¿Qué es HTML?</summary>
  <p>HTML es el lenguaje estándar
     para crear páginas web.</p>
</details>
¿Qué es HTML? (haz clic para expandir)

HTML es el lenguaje estándar para crear páginas web. Define la estructura del contenido mediante etiquetas.

¿Qué es CSS?

CSS controla la presentación visual de los documentos HTML.
