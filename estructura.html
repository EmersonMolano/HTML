Etiquetas Estructurales

<html>

Elemento raíz del documento

Definición: Es la etiqueta raíz que envuelve todo el contenido del documento HTML.

<!DOCTYPE html>
<html lang="es">
  <head>...</head>
  <body>...</body>
</html>

Todo el contenido va dentro de <html>. El atributo lang="es" indica el idioma del documento.

<header> <nav> <main> <section> <article> <aside> <footer>

Elementos semánticos de sección

Definición: Dividen la página en regiones con significado: encabezado, navegación, contenido principal, secciones, artículos, contenido lateral y pie de página.

<header>
  <h1>Mi Sitio</h1>
  <nav>
    <a href="/">Inicio</a>
    <a href="/blog">Blog</a>
  </nav>
</header>
<main>
  <section>
    <article>
      <h2>Artículo</h2>
      <p>Contenido...</p>
    </article>
  </section>
  <aside>Contenido relacionado</aside>
</main>
<footer>© 2025</footer>
<header> — Encabezado
<main> → <section> → <article>
<aside>
<footer>
