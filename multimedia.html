Etiquetas Multimedia

<img> <figure> <figcaption>

Imagen con contexto

Definición: <img> inserta imágenes. <figure> y <figcaption> dan contexto semántico relacionando la imagen con su descripción.

<figure>
  <img src="foto.jpg"
       alt="Descripción de la imagen"
       width="800" height="450"
       loading="lazy">
  <figcaption>
    Fig. 1 — Diagrama de HTML
  </figcaption>
</figure>
<video> <audio> <source>

Video y audio nativos

Definición: Permiten embeber medios sin plugins. <source> ofrece múltiples formatos para compatibilidad entre navegadores.

<video controls width="640" poster="mini.jpg">
  <source src="video.mp4"  type="video/mp4">
  <source src="video.webm" type="video/webm">
  Tu navegador no soporta HTML5 video.
</video>

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>
<details> <summary>

Contenido desplegable

Definición: Crea un acordeón interactivo sin JavaScript. <summary> es el encabezado visible; el resto se muestra u oculta al hacer clic.

<details>
  <summary>¿Qué es HTML?</summary>
  <p>HTML es el lenguaje estándar
     para crear páginas web.</p>
</details>
¿Qué es HTML? (haz clic para expandir)

HTML es el lenguaje estándar para crear páginas web. Define la estructura del contenido mediante etiquetas.

¿Qué es CSS?

CSS controla la presentación visual de los documentos HTML.
